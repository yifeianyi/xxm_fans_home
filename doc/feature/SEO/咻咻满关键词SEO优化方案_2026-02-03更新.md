#### 4.1.2 当前问题分析

**问题1：标签页使用状态管理，不利于SEO**
- **SongsPage** 使用 `useState` 管理标签切换（热歌榜、全部歌曲、原唱作品、投稿时刻）

- 搜索引擎爬虫无法索引标签页内容

- 无法为不同标签页设置独立的Meta标签

### 4.2 路由优化方案

#### 4.2.1 标签页改为独立路由

**SongsPage 优化**：
```typescript
// 优化后的路由结构
{ path: '/songs', element: <SongsPage /> }
{ path: '/songs/hot', element: <SongsPage tab="hot" /> }
{ path: '/songs/originals', element: <SongsPage tab="originals" /> }
{ path: '/songs/submit', element: <SongsPage tab="submit" /> }

// SongsPage 组件优化
const SongsPage: React.FC<{ tab?: string }> = ({ tab }) => {
    const [activeTab, setActiveTab] = useState<'hot' | 'all' | 'originals' | 'submit'>(tab || 'all');

    // 根据URL参数设置初始标签
    useEffect(() => {
        if (tab) {
            setActiveTab(tab as any);
        }
    }, [tab]);

    // 标签切换时更新URL
    const handleTabChange = (newTab: 'hot' | 'all' | 'originals' | 'submit') => {
        setActiveTab(newTab);
        if (newTab !== 'all') {
            window.history.pushState({}, '', `/songs/${newTab}`);
        } else {
            window.history.pushState({}, '', '/songs');
        }
    };

    // ...
};
```

**SEO优势**：
- 搜索引擎可以独立索引每个标签页
- 每个标签页可以设置独立的Title和Meta描述
- 用户可以直接分享特定标签页的URL
- 提高网站的可访问性和用户体验

```python
# 使用Google Indexing API
from google.oauth2 import service_account
from google.auth.transport.requests import Request
import requests

SCOPES = ["https://www.googleapis.com/auth/indexing"]

def index_to_google(url):
    """
    请求Google索引页面
    """
    credentials = service_account.Credentials.from_service_account_file(
        'service-account.json', scopes=SCOPES)
    credentials.refresh(Request())

    endpoint = "https://indexing.googleapis.com/v3/urlNotifications:publish"
    content = {
        "url": url,
        "type": "URL_UPDATED"
    }
    response = requests.post(
        endpoint,
        json=content,
        headers={
            "Content-Type": "application/json",
            "Authorization": f"Bearer {credentials.token}"
        }
    )
    return response.json()
```
