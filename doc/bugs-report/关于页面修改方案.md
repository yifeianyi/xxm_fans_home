# 关于页面修改方案

## 一、现状分析

### 1.1 当前实现
"关于"页面（AboutPage.tsx）已经实现了基本的艺人介绍功能，包括：
- 艺人基本信息展示（头像、姓名、生日、星座等）
- 声线特色展示
- 里程碑时间线
- 社交媒体链接

### 1.2 页面定位
根据用户需求，"关于咻咻"应该是**咻咻满的个人介绍页面**，展示她的个人信息、艺术特色、成长历程等。

### 1.3 后端数据支持
- **SiteSettings模型**：存储艺人基本信息、社交媒体链接
- **Milestone模型**：存储里程碑事件
- **Recommendation模型**：存储推荐语（目前未在关于页面使用）

## 二、问题与改进点

### 2.1 存在的问题

1. **页面结构不够清晰**
   - 当前各区块相对独立，缺乏整体的故事性
   - 里程碑展示方式过于简单，不够突出

2. **视觉层次需要优化**
   - 头部Hero区域不够醒目
   - 声线特色展示可以更生动

3. **内容展示可以更丰富**
   - 缺少个人简介/故事性内容
   - 缺少代表作品展示
   - 缺少粉丝互动元素

4. **社交媒体展示可以优化**
   - 当前只是简单链接，缺少平台特色
   - 可以添加粉丝数量、最近动态等信息

### 2.2 改进方向

#### 2.2.1 增强页面故事性
- 添加个人简介/自我介绍
- 优化里程碑展示，增加时间线叙事感
- 添加"我的故事"或"成长之路"板块

#### 2.2.2 丰富内容展示
- 添加代表作品/热门歌曲展示
- 添加精选演出/活动回顾
- 添加粉丝互动/感谢语

#### 2.2.3 优化视觉体验
- 增强Hero区域的视觉冲击力
- 优化排版，提高可读性
- 添加动画效果，提升交互体验

#### 2.2.4 增加互动元素
- 添加"关注我"按钮
- 添加"分享"功能
- 添加留言/祝福功能

```python
        ordering = ['-likes', '-created_at']
```

```python
# site_settings/api/views.py

class FeaturedWorksAPIView(APIView):
    """获取代表作品列表"""
    def get(self, request):
        works = FeaturedWork.objects.filter(
            is_active=True
        ).select_related('song').order_by('display_order')
        
        data = [{
            'id': w.id,
            'song': {
                'id': w.song.id,
                'name': w.song.name,
                'original_artist': w.song.original_artist,
                'cover': w.song.cover,
            },
            'display_order': w.display_order
        } for w in works]
        
        return Response(data)

class FanMessagesAPIView(APIView):
    """获取粉丝留言列表"""
    def get(self, request):
        messages = FanMessage.objects.filter(
            is_approved=True
        ).order_by('-likes', '-created_at')[:20]
        
        data = [{
            'id': m.id,
            'fan_name': m.fan_name,
            'content': m.content,
            'likes': m.likes,
            'created_at': m.created_at.isoformat()
        } for m in messages]
        
        return Response(data)
    
    def post(self, request):
        """提交留言"""
        fan_name = request.data.get('fan_name')
        content = request.data.get('content')
        
        message = FanMessage.objects.create(
            fan_name=fan_name,
            content=content
        )
        
        return Response({'id': message.id}, status=201)
```
