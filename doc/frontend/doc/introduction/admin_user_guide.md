# XXM Fans Home 后台管理页面使用指南

本文档旨在为管理员提供 XXM Fans Home 系统后台管理页面的详细使用说明。

## 1. 登录后台

1. 在浏览器中访问 `http://你的域名/admin/`
2. 输入管理员账号和密码登录

## 2. 歌曲管理 (Songs)

### 2.1 查看歌曲列表

- 进入 "Main" -> "Songs" 页面
- 可以看到歌曲列表，包含歌名、语言、歌手、最近演唱时间、演唱次数等信息
- 支持按语言、最近演唱时间进行筛选
- 支持按歌名、演唱次数、歌手进行搜索

### 2.2 管理歌曲演唱记录

- 在歌曲列表中，点击"查看记录"按钮可以查看该歌曲的所有演唱记录
- 点击记录中的日期可以跳转到对应的B站视频页面

### 2.3 合并歌曲

当存在重复歌曲时，可以进行合并操作：

1. 在歌曲列表中勾选需要合并的歌曲（至少2首）
2. 点击页面顶部的"合并选中的歌曲"操作按钮
3. 在跳转页面中选择主项（保留的歌曲）
4. 点击"合并"按钮完成操作

### 2.4 拆分歌曲

当一首歌曲包含多个不同版本的演唱记录时，可以进行拆分：

1. 在歌曲列表中勾选需要拆分的歌曲（只能选择一首）
2. 点击"拆分选中的歌曲"操作按钮
3. 在跳转页面中勾选需要拆分到新歌曲的演唱记录
4. 点击"拆分"按钮完成操作

### 2.5 批量标记语言

1. 在歌曲列表中勾选需要标记的歌曲
2. 点击"批量标记语言"操作按钮
3. 在弹出表单中输入语言名称
4. 点击"标记"按钮完成操作

### 2.6 批量添加曲风标签

1. 在歌曲列表中勾选需要添加曲风的歌曲
2. 点击"批量添加曲风标签"操作按钮
3. 在弹出表单中选择要添加的曲风
4. 点击"添加"按钮完成操作

### 2.7 批量添加标签

1. 在歌曲列表中勾选需要添加标签的歌曲
2. 点击"批量添加标签"操作按钮
3. 在弹出表单中选择要添加的标签
4. 点击"添加"按钮完成操作

## 3. 演唱记录管理 (SongRecords)

### 3.1 查看演唱记录

- 进入 "Main" -> "Song records" 页面
- 可以看到所有演唱记录，包含歌曲名、演唱时间、视频链接、封面缩略图等信息
- 支持按演唱时间、歌曲名进行筛选
- 支持按歌曲名、备注进行搜索

### 3.2 添加演唱记录

1. 点击页面右上角的"添加歌曲记录"按钮
2. 填写表单信息：
   - 歌曲：选择或输入歌曲名称
   - 演唱时间：选择日期
   - 视频链接：输入B站视频链接
   - 封面链接：输入封面图片链接
   - 备注：输入相关备注信息
3. 点击"保存"按钮完成添加

### 3.3 从BV号导入演唱记录

1. 点击页面右上角的"从BV导入"按钮
2. 在弹出表单中输入B站视频BV号
3. 如果系统能自动匹配到歌曲，会自动完成导入
4. 如果无法匹配，需要手动选择对应的歌曲
5. 导入完成后会显示导入结果

### 3.4 替换封面图

1. 在演唱记录列表中点击要修改的记录
2. 在编辑页面找到"替换封面"字段
3. 上传新的封面图片
4. 点击"保存"按钮完成操作

## 4. 曲风管理 (Styles)

### 4.1 查看曲风列表

- 进入 "Main" -> "Styles" 页面
- 可以看到所有曲风标签
- 支持按名称进行搜索

### 4.2 添加曲风

1. 点击页面右上角的"添加曲风"按钮
2. 输入曲风名称
3. 点击"保存"按钮完成添加

### 4.3 批量为歌曲添加曲风

1. 进入 "Main" -> "Song styles" 页面
2. 点击页面右上角的"批量添加歌曲曲风"按钮
3. 在跳转页面中：
   - 在左侧列表中搜索并选择歌曲
   - 点击"添加到右侧"按钮将歌曲添加到右侧列表
   - 选择要添加的曲风
4. 点击"保存"按钮完成操作

## 5. 标签管理 (Tags)

### 5.1 查看标签列表

- 进入 "Main" -> "Tags" 页面
- 可以看到所有标签
- 支持按名称进行搜索

### 5.2 添加标签

1. 点击页面右上角的"添加标签"按钮
2. 输入标签名称
3. 点击"保存"按钮完成添加

### 5.3 为标签批量标记歌曲

1. 在标签列表中勾选要操作的标签
2. 点击"为选中的标签批量标记歌曲"操作按钮
3. 在跳转页面中：
   - 在左侧列表中搜索并选择歌曲
   - 点击"选择"按钮将歌曲添加到右侧列表
4. 点击"保存"按钮完成操作

### 5.4 批量为歌曲添加标签

1. 进入 "Main" -> "Song tags" 页面
2. 点击页面右上角的"批量添加歌曲标签"按钮
3. 在跳转页面中：
   - 在左侧列表中搜索并选择歌曲
   - 点击"添加到右侧"按钮将歌曲添加到右侧列表
   - 选择要添加的标签
4. 点击"保存"按钮完成操作

## 6. 推荐语管理 (Recommendations)

### 6.1 查看推荐语列表

- 进入 "Main" -> "Recommendations" 页面
- 可以看到所有推荐语，包含内容预览、是否激活、更新时间、推荐歌曲数量等信息
- 支持按是否激活、更新时间进行筛选
- 支持按内容进行搜索

### 6.2 添加推荐语

1. 点击页面右上角的"添加推荐语"按钮
2. 输入推荐语内容
3. 从右侧列表中选择推荐的歌曲（可选）
4. 设置是否激活
5. 点击"保存"按钮完成添加

## 7. 粉丝二创作品管理 (FansDIY)

### 7.1 合集管理

#### 7.1.1 查看合集列表

- 进入 "Fans diy" -> "Collections" 页面
- 可以看到所有合集，包含名称、作品数量、位置、显示顺序等信息

#### 7.1.2 添加合集

1. 点击页面右上角的"添加合集"按钮
2. 输入合集名称、位置、显示顺序等信息
3. 点击"保存"按钮完成添加

### 7.2 作品管理

#### 7.2.1 查看作品列表

- 进入 "Fans diy" -> "Works" 页面
- 可以看到所有作品，包含标题、作者、合集、位置、显示顺序、封面预览、观看链接等信息
- 支持按合集、作者进行筛选
- 支持按标题、作者、合集名称、备注进行搜索

#### 7.2.2 添加作品

1. 点击页面右上角的"添加作品"按钮
2. 填写表单信息：
   - 合集：选择所属合集
   - 标题：输入作品标题
   - 作者：输入作者名称
   - 位置：输入位置信息
   - 显示顺序：输入显示顺序
   - 封面链接：输入封面图片链接
   - 观看链接：输入观看链接
   - 备注：输入相关备注信息
3. 点击"保存"按钮完成添加

#### 7.2.3 从BV号导入作品

1. 点击页面右上角的"从BV导入"按钮
2. 在弹出表单中输入B站视频BV号
3. 输入合集名称（可选）
4. 输入备注信息（可选）
5. 点击"导入"按钮完成操作

## 8. 网站设置 (Site settings)

### 8.1 查看/修改网站图标

- 进入 "Main" -> "Site settings" 页面
- 可以看到当前网站图标预览
- 点击"修改"按钮可以上传新的网站图标
- 上传后系统会自动将图片转换为favicon.ico格式
